# CarPhotoManager

אפליקציית אנדרואיד לניהול תמונות רכבים לפי מספר לוחית רישוי, עם OCR מקומי (ML Kit),
מסד נתונים מקומי (Room) וגלריה מהירה גם עם מאות אלפי תמונות.

## איך לבנות APK בלי אנדרואיד סטודיו (מהפלאפון)

1. צור ריפו חדש בחשבון ה-GitHub שלך (למשל `car-photo-manager`).
2. העלה את כל התוכן של התיקייה הזו לריפו (אפשר דרך אפליקציית GitHub, או דרך דפדפן –
   Upload files, גרור את כל התיקיות והקבצים).
3. וודא שהתיקייה `.github/workflows/build.yml` הועלתה כמו שהיא (זה מה שמפעיל את הבנייה).
4. עבור ל-Actions בריפו → תראה ריצה בשם "Build APK" מתחילה אוטומטית.
5. אחרי כ-2-4 דקות, כשה-workflow מסתיים בהצלחה, כנס לריצה → למטה יש "Artifacts" →
   הורד את `app-debug-apk` (קובץ zip עם ה-APK בפנים).
6. פתח את ה-zip בטלפון, תתקין את ה-APK (יידרש לאשר "התקנה ממקורות לא ידועים" בפעם הראשונה).

אפשר גם להריץ בנייה ידנית בלי push: Actions → Build APK → Run workflow.

## מבנה הפרויקט
- `app/src/main/java/.../data` — Room (Car, Photo, DAOs, Repository)
- `app/src/main/java/.../ocr` — זיהוי לוחית רישוי (ML Kit, אופליין)
- `app/src/main/java/.../ui` — מסכי Compose (בית, צילום, גלריה, צפייה מלאה)
- `app/src/main/java/.../work` — סריקה הדרגתית של גלריית המכשיר ברקע

## הערות
- זהו build.gradle ל-debug בלבד (לצורך בדיקות). לפני הפצה רשמית יידרש חתימת release.
- OCR רץ לגמרי במכשיר (ML Kit on-device) — האפליקציה עובדת ללא אינטרנט.
